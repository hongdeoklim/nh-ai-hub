# 7️⃣ 영상 비서 (Video Assistant)

이 비서는 입력된 텍스트나 주제를 바탕으로 15초~30초 분량의 숏폼 영상 대본(화면 프롬프트 + 자막)을 자동으로 기획해 줍니다.

## ⚙️ 사용 방법

1. 터미널에서 스크립트를 실행하여 영상 대본과 씬(Scene) 프롬프트를 추출합니다.
```bash
python video_assistant.py
```
2. 생성된 대본을 기반으로 Higgsfield, HeyGen, 혹은 타 비디오 AI 생성기에 붙여넣어 영상을 제작합니다.

*(추후 Higgsfield API가 정식으로 지원/결제되면, 파이썬 파일 내의 주석 처리된 `requests` 부분을 활성화하여 API 다이렉트 호출이 가능합니다.)*
