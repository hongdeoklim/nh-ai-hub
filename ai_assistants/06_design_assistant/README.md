# 6️⃣ 디자인 비서 (Design & Canva Assistant)

이 비서는 텍스트 아이디어를 이미지 생성이나 디자인 도구(Canva)에 알맞은 형태의 프롬프트 및 구조로 변환해 주는 역할을 합니다.

## ⚙️ 사용 방법 (로컬 기반)

외부 API 키 없이, 저(메인 AI)의 **이미지 생성 기능(`generate_image`)** 과 연동하여 사용하도록 설계되었습니다.

1. 스크립트를 실행하여 기획안을 프롬프트로 변환합니다.
```bash
python design_assistant.py
```
2. 생성된 프롬프트를 복사하거나, 바로 저에게 이렇게 명령하세요.
> "방금 만든 디자인 비서의 프롬프트를 바탕으로 이미지를 생성해줘!"

*(참고: 실제 Canva 자동화가 필요하다면 Canva Developers API를 신청해야 하지만, 1인 사업가 수준에서는 메인 AI의 이미지 생성 기능을 활용하는 것이 가장 빠르고 효율적입니다.)*
